# My portfolio [felixhaeberle.com](https://felixhaeberle.com) 🤟

Have fun exploring my personal site and leave a comment or feedback if you want to! 🤙

Built with **Nextjs**, **Sanity**, **Styled Components**, **IBM Carbon Icons** & more.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm -w run dev

# Build for production
pnpm -w run build
```

## Studio (Sanity CMS)

Sanity Studio is located in the `studio` directory.

```bash
# Navigate to studio directory
cd studio

# Start Sanity Studio
pnpm run start

# Build Sanity Studio
pnpm run build
```
